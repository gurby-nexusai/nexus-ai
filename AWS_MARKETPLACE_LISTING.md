## Deployment Instructions

### One-Click Deployment
1. Subscribe to Nexus AI on AWS Marketplace
2. Choose your pricing tier
3. Click "Launch CloudFormation Stack"
4. Select parameters:
   - EC2 Key Pair (for SSH access)
   - LLM Provider (Ollama, Groq, Nova, etc.)
   - SSH access IP range
5. Launch stack (takes 10-15 minutes)
6. Access application via Load Balancer URL
7. Login with: admin@example.com (any password)

### Post-Deployment Configuration
1. Configure LLM settings in application (⚙️ icon)
2. Create user accounts
3. Upload knowledge base documents (optional)
4. Customize applications as needed

## Next Steps for AWS Marketplace Listing

1. **Create AWS Marketplace Seller Account**
   - Go to AWS Marketplace Management Portal
   - Complete seller registration
   - Provide tax information (W-9 or W-8)
   - Set up bank account for payments

2. **Upload CloudFormation Template**
   - Use cloudformation-template.yaml
   - Test deployment in your own AWS account first
   - Validate all resources create successfully

3. **Create Product Listing**
   - Fill in all product information above
   - Upload screenshots and logo
   - Set pricing tiers
   - Enable 7-day free trial

4. **Submit for Review**
   - AWS reviews listing (typically 2-4 weeks)
   - Address any feedback
   - Get approval

5. **Go Live**
   - Product appears in AWS Marketplace
   - Start receiving customer subscriptions
   - AWS handles billing and payments
